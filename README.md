# 결제 기반 크레딧을 충전하고, 크레딧을 소모해 ML 예측이 가능한 백엔드 서비스.

## 사용 프레임워크
- Django Rest FrameWork, FastAPI(머신러닝 추론), MySQL

## 설치방법
  1. git clone https://github.com/KimGyeongMinB/practice_repository.git
  2. cd practice_repository
  3. python -m venv venv # 가상환경
  4. source venv/bin/activate      # macOS / Linux
  5. venv\Scripts\activate         # Windows
  6. pip install -r requirements.txt
